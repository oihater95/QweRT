### ✅ Commit Convention

#### 한 번에 git add . 하는 것보다 commit type에 맞게 분리하는 걸 지향합니다.
```
docs : README.md 등 문서 작성 및 수정
code : 코드 작성
fix : 코드 수정
merge : merge request할 때
```
git convention : commit message

<br>
